TICKET TITLE:
Device Assigned 169.254 IP Address

CATEGORY:
Network Connectivity

PRIORITY:
Medium

ENVIRONMENT:
Operating System: Windows 10
Device Type: Laptop
Network Type: Wired and Wireless
 Location: Office environment

--------------------------------

USER ISSUE:
User reports being unable to access the network. Investigation shows the device has been assigned an IP address in the 169.254.x.x range.
---------------------------------

INITIAL CHECKS:
- Confirmed Wi-Fi icon status
- Asked when issue began
- Confirmed no recent system changes
- Confirmed network issue persisted after reconnecting

--------------------------------

TROUBLESHOOTING STEPS:
1. Confirmed the device was connected to the correct network.
2. Checked network cable and Wi-Fi connection status.
3. Verified the device IP address using `ipconfig`.
4. Confirmed the device was unable to obtain an IP address from the DHCP server.
5. Released and renewed the IP address using `ipconfig /release` and `ipconfig /renew`.
6. Restarted the device.
7. Tested connectivity using a known working Ethernet cable.
8. Confirmed other users were able to connect to the network.

--------------------------------

RESOLUTION:
The issue was escalated after identifying that the device could not obtain a valid IP address from the DHCP server.

--------------------------------

OUTCOME:
Ticket was escalated to the network team for further investigation. User was informed of the status and next steps.
--------------------------------

TICKET STATUS:
Escalated
