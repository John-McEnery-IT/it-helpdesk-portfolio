TICKET TITLE:
Connected to Wi-Fi but No Internet Access

CATEGORY:
Network Connectivity

PRIORITY:
Medium

ENVIRONMENT:
Operating System: Windows 10
Device Type: Laptop
Network Type: Wireless (Wi-Fi)
 Location: Office environment

--------------------------------

USER ISSUE:
User reports that their device is connected to the Wi-Fi network, but they are unable to access the internet.
---------------------------------

INITIAL CHECKS:
- Confirmed Wi-Fi icon status
- Asked when issue began
- Confirmed no recent system changes

--------------------------------

TROUBLESHOOTING STEPS:
1. Confirmed the device showed as connected to the correct Wi-Fi network.
2. Asked the user to test multiple websites to rule out a single-site issue.
3. Checked whether other users were experiencing the same issue.
4. Verified the device had a valid IP address.
5. Ran basic network troubleshooting using Windows Network Diagnostics.
6. Asked the user to restart the device.
7. Power-cycled the router and access point.

--------------------------------

RESOLUTION:
The issue was resolved after power-cycling the router, restoring internet connectivity to the network.

--------------------------------

OUTCOME:
User confirmed that internet access was restored and normal browsing functionality resumed.
--------------------------------

TICKET STATUS:
Resolved
