# Ticket 1 – Unable to Connect to Wi-Fi

## Issue Summary
User reports being unable to connect to the office Wi-Fi network. The network is visible, but the device fails to connect.

##Category:
Network Connectivity
##Priority:
Medium

## User Environment
- Device: Laptop
- Operating System: Windows 10
- Network Type: Wireless (Wi-Fi)
- Location: Office environment

## Troubleshooting Steps Taken
1. Confirmed the Wi-Fi network name (SSID) was correct.
2. Verified that Airplane Mode was turned off and Wi-Fi was enabled.
3. Asked the user to forget the Wi-Fi network and reconnect.
4. Confirmed the correct Wi-Fi password was being entered.
5. Confirmed whether the user had recently changed their Wi-Fi password.
6. Restarted the user’s laptop.
7. Checked that other users were able to connect to the same network.
8. Verified the wireless adapter was enabled in Network Settings.

## Resolution
The issue was resolved after restarting the device and reconnecting to the Wi-Fi network using the correct credentials.

## Outcome
User confirmed internet access was restored and Wi-Fi connection was stable. No further issues reported.
